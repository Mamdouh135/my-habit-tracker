# Habit Tracker SaaS Backend

This is the backend for a multi-tenant (SaaS) habit tracker application. Each user is isolated by their user account. Built with Node.js, Express, and SQLite.

## Features
- User registration and login (to be implemented)
- CRUD operations for habits (to be implemented)
- Multi-tenant: each user's data is isolated

## Getting Started
1. Install dependencies: `npm install`
2. Start the server: `npm start`
